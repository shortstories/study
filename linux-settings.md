# 유용했던 리눅스 세팅

## Terminal

* guake + byobu
  * guake: https://github.com/Guake/guake
  * byobu: http://byobu.co/index.html
* * F12만 누르면 언제나 터미널이 띄워지고 분할도 자유자재
  * guake의 단축키를 좀 많이 삭제해야됨
  * byobu의 스크롤을 쓰고 싶으면 Alt + F12로 Mouse: ON 할 것
